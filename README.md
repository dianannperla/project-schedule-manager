# Project Schedule Manager

## Giới thiệu
Project Schedule Manager là một ứng dụng giúp người dùng quản lý lịch trình hàng ngày. Người dùng có thể thêm, xóa và chỉnh sửa các cuộc hẹn và công việc một cách dễ dàng. Ứng dụng này cung cấp giao diện thân thiện và dễ sử dụng, cho phép người dùng theo dõi thời gian biểu của mình một cách hiệu quả.

## Tính năng
- Thêm, chỉnh sửa và xóa cuộc hẹn
- Xem lịch trình hàng ngày, hàng tuần
- Nhắc nhở cuộc hẹn qua thông báo
- Tích hợp với lịch Google
- Tìm kiếm và lọc cuộc hẹn theo từ khóa

## Công nghệ sử dụng
- React.js
- Redux cho quản lý trạng thái
- Material-UI cho giao diện
- Firebase cho lưu trữ và xác thực

## Cài đặt
1. Clone repository này.
2. Cài đặt các gói cần thiết bằng cách chạy `npm install`.
3. Khởi động ứng dụng bằng lệnh `npm start`.

## Đóng góp
Chúng tôi luôn chào đón sự đóng góp từ cộng đồng! Nếu bạn muốn đóng góp vào dự án này, hãy tạo một pull request hoặc mở issue để thảo luận.

Nếu bạn có bất kỳ câu hỏi nào, vui lòng liên hệ với chúng tôi qua email hoặc tạo issue trên GitHub.
